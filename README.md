# Online-food-delivery-system
基于 Spring Boot + Vue 的在线外卖系统，是一个集订餐、送餐服务于一体的平台。它包含餐厅展示、菜品浏览、下单、支付、订单追踪等功能。前端 Vue 提供用户友好的界面，方便顾客操作；后端 Spring Boot 确保系统稳定，保障订单数据和用户信息的安全存储和高效处理，为用户提供便捷的在线外卖体验，同时助力商家拓展业务。
